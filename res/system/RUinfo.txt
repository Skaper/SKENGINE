SKEGINE 0.1.12\n
Пакет "game" - место для создания игры.
Каждая сцена создается в отдельном пакете внутри scenes,
например: game/scenes/ExampleScene
ScenesController - предназначен для управления сценами
GameManager - для запуска игры, изменять его не следует.

Каждый новый игровой объект должен наследоваться от класса GameObject.
Каждый новая сцена должена наследоваться от класса Scene.


Игровые ресурсы res:
animations/ - папка для анимаций, каждый набор анимаций должен быть в отдельной папке
audio/ - Папка для звуковых фалов формата .wav
images/ - Папка для изображений, формата .jpg или .png
system/config.properties - настройка движка/игры
